# JHack
JavaHack Simulator is an ongoing project inspired by the game "GreyHack," simulating a Linux environment for educational purposes. 
The project, currently in development, aims to recreate the Linux system found in GreyHack using Java. While it does not yet feature simulated hacking attacks, they are planned for future updates. 
The primary goal of this project is to facilitate learning by providing users with a safe environment to explore and understand hacking techniques.

Planned Features:

- Simulated Linux Environment: Experience a realistic Linux system simulation within Java.
- Learning Platform: Use the simulator as an educational tool to learn and practice hacking skills.
- User-Friendly Interface: Intuitive interface designed for ease of use and accessibility.
- Secure Environment: Practice hacking in a secure and controlled virtual environment, ensuring user safety.
